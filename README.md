
# Windows Party Mode

Welcome to windows party mode!
 
 Features
 -
Psychedelic background
Mouse Cursor movement
Simple source code
Python Usage




## Installation

Download and install dependencies :

```bash
pip install pygame pyautogui

```
Check dependencies : 

```bash
pip show pygame pyautogui

```
